Instrucciones de Prueba

  • Verifica el estado: El punto indica si la API está conectada
  • Haz clic en el chatbot: Para abrir/cerrar la ventana de chat
  • Escribe preguntas: Prueba con temas de panadería, masa madre, productos, etc.
  • Prueba soporte humano: Usa el botón "💬 Hablar con alguien"
  • Da feedback: Usa 👍 o 👎 después de las respuestas
  • Verifica localStorage: Abre DevTools → Application → Local Storage
